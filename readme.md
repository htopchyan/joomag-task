# Image Form Tag Retriever

Image Form Tag Retriever is a small program in Python that is capable of scanning a web page at a
    given URL and retrieve 
    
        1. Number of html form elements (with method="get") present in that page
        2. Number of HTML image tags present in that page


### Prerequisites


This project uses "requests" and "lxml" packages



### Installing
```
pip install requests
pip install lxml
```

## Version
Python 3
